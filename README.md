

## Description

*Wallet Coding Challenge*


## Developer Guide

### Installation Dependencies

```bash
$ npm install
```

## Running the application

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# debug mode
$ npm run start:debug

# production mode
$ npm run start:prod
```

## Testing the application

```bash
# unit tests
$ npm run test:unit

# Unit Test Coverage
$ npm run unit-test:cov

# e2e tests
$ npm run test:e2e

```

## Setup application 

```
docker-compose up -d

```
### Access the api
Please use the below server url to access the api server 

```
 localhost:3000/wallet/1

```
